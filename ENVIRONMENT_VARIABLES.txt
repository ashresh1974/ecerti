Copy and paste these into your Vercel project settings:

DB_DATABASE=ecertificate
EMAIL_USER=mguecertificate@gmail.com

FRONTEND_URL=https://ecerti.vercel.app
NODE_ENV=production

INSTRUCTIONS:
=============

1. Go to https://vercel.com/dashboard
2. Select your ecerti project
3. Go to Settings → Environment Variables
4. Add each variable above
5. Save and redeploy

IMPORTANT NOTES:
================

- DB_HOST: If using local MySQL, ensure it's accessible from internet
  For production, use a managed database service like PlanetScale
  
- EMAIL_PASS: This is a Gmail App Password, not your regular password
  Generate one at: https://myaccount.google.com/apppasswords
  
- SESSION_SECRET: Change this to a random string for production
  Generate one: node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"
  
- FRONTEND_URL: Will be your Vercel deployment URL
  Default: https://ecerti.vercel.app
  Or your custom domain if configured

DATABASE SETUP:
===============

Before deploying, ensure your MySQL database is set up:

1. Create database:
   CREATE DATABASE ecertificate;

2. Create tables (check backend/src/config/db.js for schema)

3. For production, use a managed database service:
   - PlanetScale (MySQL-compatible)
   - AWS RDS
   - DigitalOcean
   - Google Cloud SQL
   - Azure Database for MySQL

GMAIL APP PASSWORD:
===================

1. Enable 2-Factor Authentication on your Gmail account
2. Go to https://myaccount.google.com/apppasswords
3. Select "Mail" and "Windows Computer"
4. Generate app password
5. Use the generated password as EMAIL_PASS

FILE STORAGE:
=============

The app currently stores files locally. For production, migrate to:
- AWS S3
- Google Cloud Storage
- Azure Blob Storage
- Vercel Blob Storage

Update backend/src/controllers/certificateController.js to use cloud storage.

DEPLOYMENT STEPS:
=================

1. Go to https://vercel.com/new
2. Import your GitHub repository (ashresh1974/ecerti)
3. Configure environment variables (use values above)
4. Click Deploy
5. Wait for build to complete
6. Your app will be live at https://ecerti.vercel.app

MONITORING:
===========

After deployment:
1. Check Vercel dashboard for build status
2. View runtime logs for any errors
3. Test API endpoints
4. Verify email notifications work
5. Test certificate generation
